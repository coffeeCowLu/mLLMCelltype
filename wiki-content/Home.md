# mLLMCelltype Wiki

Welcome to the knowledge base for **mLLMCelltype**, a multi-LLM consensus framework for single-cell RNA sequencing cell type annotation.

## Overview

mLLMCelltype combines predictions from multiple large language models to annotate cell types in scRNA-seq data.
